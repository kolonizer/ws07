Правила работы с выполненным проектом
    • Скачать http://LOVE2D.org и установить в C:\LOVE2D\
    • Скачайте проект мастерской из GitHub SSYP19 WS07
    • Скопируйте проект в папку с установленным LOVE2D
    • Запускайте единственный в корневой папке файл start.bat
    • Инструкция по управлению появится на экране

Клавиши управления
	W A S D - управление
	space - атака
	q - собрать предмет
	TAB - Инвентарь / Мини-карта
